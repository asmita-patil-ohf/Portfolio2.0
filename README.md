# 🌐 Asmita Portfolio 2.0
> Crafted with 💜 and lots of debugging.
> 
A modern and responsive personal portfolio website built with **React**, **Vite**, **Tailwind CSS**, and **Framer Motion**. Designed with a futuristic dark aesthetic featuring glassmorphism, animated particles, and smooth transitions.

## ✨ Features

* 🎨 Modern UI with Purple/Fuchsia theme
* ⚡ Built with React + Vite
* 📱 Fully Responsive Design
* ✨ Smooth Animations using Framer Motion
* 🌌 Animated Particles Background
* 🛠️ Skills and Tech Stack Section
* 📂 Project Showcase
* 💻 Coding Profiles
* 📞 Contact Section
* 📄 Resume Download

## 🛠️ Tech Stack

* React
* Vite
* Tailwind CSS
* Framer Motion
* React Icons
* tsParticles
